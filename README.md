- 📱 Fully responsive design
- 🎨 Modern and clean UI
- 👨‍💻 Projects showcase section

## 🚀 How to Run

1. Clone the repo

```bash
   git clone https://github.com/Muhammad-Qasim-Ali/portfolio.git
```

2. Install dependencies

```bash
   npm install
```

3. Run the project

```bash
   npm start
```

4. Open browser at `http://localhost:3000`
